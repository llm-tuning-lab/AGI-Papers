# 선형 RNN 병렬화 속도와 표현력 분석

논문: **"Why Are Linear RNNs More Parallelizable?"**

이 논문은 선형 RNN, 비선형 RNN, 트랜스포머의 병렬 처리 특성과 표현력 사이의 트레이드오프를 수학적인 **'회로 깊이(Circuit Depth)'** 클래스로 완벽히 증명한 수작입니다.

### 1. 아키텍처별 회로 깊이와 속도
병렬화 연산에서 벽시계 시간(Wall-clock time)은 회로의 깊이에 비례합니다.
- **트랜스포머 (TC_0)**: $O(\log n)$ (수치 환산 시 깊이 16~20) - 얕은 깊이로 대규모 병렬 압도.
- **최신 선형 RNN (RWKV-7, DeltaNet, PNC_1)**: $O(\log n \log^* n)$ - 트랜스포머와 유사한 엄청난 스피드 유지.
- **비선형 RNN (sLSTM 등)**: $O(\log^2 n)$ (수치 환산 시 깊이 256~400) - 병렬 처리 시 최대 20배 속도 저하 발생.

### 2. 지능/표현력 (Expressivity)의 트레이드오프
속도가 빠른 만큼 지능의 손실이 따릅니다. 트랜스포머와 초기 선형 모델(Mamba)은 '반복 행렬 곱셈' 같은 대수학 구조 문제에서 한계를 드러냈습니다.
하지만 DPLR 계열(RWKV-7, DeltaNet)은 훌륭한 속도를 유지하면서도 복잡도 PNC_1 수준에 도달하여, 훨씬 똑똑한 추론력을 보여주는 이상적 타협점입니다.

### 3. xLSTM의 본질
Sepp Hochreiter 교수의 평가처럼, xLSTM은 속도가 압도적인 **선형 mLSTM**과, 극단적으로 복잡한 세계 모델 및 추론을 수행하는 **비선형 sLSTM**을 모두 탑재한 구조입니다. 즉, 병렬성과 표현력 양극단의 한계를 해결할 수 있는 '완전체'로서의 가능성을 시사합니다.
