# MDM-Prime-v2: 확산 언어 모델 강화

일반적으로 자기회귀 모델(ARM)에 밀려 효율성이 16배 낮은 **마스크 확산 언어 모델(MDM)**을 이긴 아키텍처 개선 논문입니다. 변분 상한 최적화를 위해 다음과 같은 방법을 적용했습니다.

### 1. 도입 기술
- **이진 인코딩 (Binary Encoding)**: 부분 토큰 마스킹 길이를 조절. 토큰을 쪼개어 엔트로피를 활용하는 이진 인코딩 채택.
- **인덱스 셔플링 (Index Shuffling)**: 기존 토크나이저(BPE)의 인덱스 편향으로 엔트로피가 극단적으로 낮아지는 문제를 랜덤 테이블을 이용한 셔플링으로 이론적 최대치에 가깝게 회복시켰습니다.

### 2. 연산 효율성 극대화 (21.8배)
결과적으로 MDM-Prime-v2는 동일 연산 예산 내에서 ARM의 퍼포먼스를 상회하였으며, 오히려 데이터 양의 확장이 모델 파라미터 크기를 늘리는 것(Chinchilla 스케일링)보다 더 유리하다는 신규 스케일링 법칙 체계를 시사합니다.
